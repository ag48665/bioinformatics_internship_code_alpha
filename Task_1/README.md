\# 🧬 Task 1 – DNA/Protein Sequence Analysis Using BLASTP



\## 📌 Overview



This project was completed as part of the \*\*CodeAlpha Bioinformatics Internship\*\*.



The objective was to analyze the \*\*human hemoglobin beta (HBB) protein\*\* using the \*\*NCBI BLASTP\*\* tool to identify homologous protein sequences and evaluate sequence similarity across different species.



\---



\## 🎯 Objectives



\- Retrieve a protein sequence from UniProt.

\- Perform a BLASTP search against a protein database.

\- Identify homologous protein sequences.

\- Analyze sequence identity, similarity, alignment score, and E-value.

\- Interpret the biological significance of the results.



\---



\## 🛠️ Tools Used



\- UniProt

\- NCBI BLASTP

\- ClusteredNR Database



\---



\## 📊 Results



The BLASTP analysis identified multiple homologous hemoglobin beta proteins from different mammalian species. The best matches showed:



\- \*\*Query Coverage:\*\* 100%

\- \*\*Highest Identity:\*\* 93.20%

\- \*\*Lowest E-value:\*\* 2e-97

\- \*\*Best Alignment Score:\*\* 288 bits



These results demonstrate that the hemoglobin beta protein is highly conserved among mammals, reflecting its essential role in oxygen transport.
