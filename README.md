# 🧬 CodeAlpha Bioinformatics Internship

This repository contains the projects completed during the **CodeAlpha Bioinformatics Internship**. The internship focused on applying bioinformatics tools to analyze protein sequences, compare evolutionary relationships, and explore protein structures.

---

## 📋 Internship Tasks

### ✅ Task 1 – DNA/Protein Sequence Analysis Using BLASTP

**Objective**
- Retrieve the human hemoglobin beta (HBB) protein sequence.
- Perform a BLASTP search using the NCBI BLAST tool.
- Identify homologous protein sequences.
- Analyze sequence identity, alignment score, query coverage, and E-value.

**Tools**
- UniProt
- NCBI BLASTP
- ClusteredNR Database

**Key Results**
- Query Coverage: **100%**
- Highest Identity: **93.20%**
- Lowest E-value: **2e-97**
- Best Alignment Score: **288 bits**

---

### ✅ Task 2 – Multiple Sequence Alignment

**Objective**
Perform a Multiple Sequence Alignment (MSA) of five hemoglobin beta protein sequences to identify conserved amino acid residues and investigate evolutionary relationships.

**Species**
- Homo sapiens
- Mus musculus
- Canis lupus familiaris
- Bos taurus
- Sus scrofa

**Tools**
- UniProt
- Clustal Omega

**Key Results**
- Identification of highly conserved amino acid residues.
- Construction of Guide Tree and Phylogenetic Tree.
- Comparative analysis of mammalian hemoglobin beta proteins.

---

### ✅ Task 3 – Protein Structure Prediction

**Objective**
Predict and visualize the three-dimensional structure of the human hemoglobin beta protein using AlphaFold.

**Protein**
- Hemoglobin subunit beta (HBB)
- UniProt ID: P68871
- Organism: Homo sapiens

**Tools**
- UniProt
- AlphaFold Protein Structure Database

**Key Results**
- Visualization of the characteristic globin fold.
- High-confidence AlphaFold prediction.
- Structural analysis of the HBB protein.
