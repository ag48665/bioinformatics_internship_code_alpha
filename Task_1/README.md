# 🧬 Task 1 – DNA/Protein Sequence Analysis Using BLASTP

## 📌 Overview

This project was completed as part of the **CodeAlpha Bioinformatics Internship**.

The objective of this task was to analyze the **human hemoglobin beta (HBB) protein** using the **NCBI BLASTP** tool. The analysis aimed to identify homologous protein sequences and evaluate sequence similarity across different species.

---

## 🎯 Objectives

* Retrieve the HBB protein sequence from **UniProt**.
* Perform a **BLASTP** search against the ClusteredNR protein database.
* Identify homologous protein sequences.
* Compare sequence identity, query coverage, alignment score, and E-value.
* Interpret the biological significance of the obtained results.

---

## 🛠️ Tools & Resources

* **UniProt**
* **NCBI BLASTP**
* **ClusteredNR Database**

---

## 📊 Results

The BLASTP search identified multiple homologous **hemoglobin beta proteins** from various mammalian species.

### Key Findings

| Metric               |        Value |
| -------------------- | -----------: |
| Query Coverage       |     **100%** |
| Highest Identity     |   **93.20%** |
| Lowest E-value       |    **2e-97** |
| Best Alignment Score | **288 bits** |

### Interpretation

The results indicate that the **hemoglobin beta (HBB) protein is highly conserved among mammals**, highlighting its critical biological function in oxygen transport and evolutionary conservation.
